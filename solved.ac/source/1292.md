
# 1292번 쉽게푸는 문제
[문제 보러가기](https://www.acmicpc.net/problem/1292)

## 🅰 설계
구간의 시작과 구간의 합을 입력받아서 구간의 합을 구하자.
수열이 정해져 있으니까 그 수열을 만들어주었고, 그 수열에서 범위를 따서 그냥 더해주었다.


> python

```py


a,b = map(int, input().split())
temp = []
for i in range(1, b + 1):
    for j in range(i):
        temp.append(i)
    if len(temp) > b:
        break

temp = temp[a- 1: b]


print(sum(temp))
```
